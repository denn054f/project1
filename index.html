Hvad bruges det til?

JavaScript er meget populært, da det er meget anvendeligt i forskellige sammenhænge. Det kan bl.a. bruges til at skabe app- og hjemmesideapplikationer, bygge hjemmesideservere eller til at designe browserspil. Men vigtigst af alt kan sproget bruges til at skabe interaktive hjemmesider, ligesom at vi kan sætte en onclick til knappen herunder.


      <button type="button" name="button" onclick="alert('Hello World!')">Klik her!</button>

Langt de fleste hjemmesider benytter JavaScript på en eller anden måde. På internettet bruges JavaScript bl.a. til at gøre hjemmesider dynamiske og interaktive, f.eks. Med pop-up notifikationer, udregnere, tællere mm.


      <!DOCTYPE html>
      <html>

      <head>
          <meta charset="UTF-8">
          <meta name="viewport" content="width=device-width, initial-scale=1.0">
          <title>Hvad er Javascript?</title>
      </head>

      <body>

          <!-- Dit første Javascript kan indsættes her! -->
          <button type="button" name="button" onclick="alert('Hello World!')">Klik her!</button>

      </body>

      </html>
