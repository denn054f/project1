Introduktion

JavaScript er meget populært, da det er meget anvendeligt i forskellige sammenhænge. Det kan bl.a. bruges til at skabe app- og hjemmesideapplikationer, bygge hjemmesideservere eller til at designe browserspil. På internettet bruges JavaScript bl.a. til at gøre hjemmesider dynamiske og interaktive, f.eks. Med pop-up notifikationer, udregnere, tællere mm. Men vigtigst af alt kan sproget bruges til at skabe interaktive websider.

HTML5 & CSS

Før du giver dig i kast med JavaScript for at skabe interaktive hjemmesider, er det en fordel at have styr på, eller i hvert fald kendskab til, grundlæggende webudvikling med HTML og CSS. Det er ikke nødvendigt at kunne skrive CSS for at lære og forstå JavaScript, men det er en god idé at vide, hvad det er og hvad man kan med det. Kendskabet til HTML er lidt vigtigere, da det er HTML-elementer som der manipuleres med i JavaScript.
